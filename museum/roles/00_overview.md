# Музейное агентство — Обзор

Агентство создаёт концепции иммерсивных музейных опытов: временные и постоянные выставки, аудиогиды, интерактивные инсталляции, образовательные программы. Мы не декорируем залы — мы проектируем опыт, который посетитель переживает, запоминает и хочет пережить снова.

Клиент приходит с коллекцией и идеей. Уходит с готовой концепцией выставки, сценарием опыта и техническим заданием для реализации.

## Правила работы агентства

1. **Концепция первична.** Ни один элемент выставки не создаётся без утверждённой концептуальной рамки.
2. **Куратор — точка входа и выхода.** Все коммуникации с клиентом идут через него.
3. **Art Historian и Experience Designer работают параллельно** после получения Creative Brief.
4. **Mediator собирает Музейный Дизайн-документ** из материалов всех специалистов.
5. **Museum Researcher работает автономно** — его отчёты доступны команде, но не блокируют основной процесс.
6. **Документ — живой.** МДД обновляется по ходу проекта. Changelog обязателен.

## Что получает клиент

- **Концепция выставки** — главная идея, нарративная структура, логика пространства
- **Сценарий опыта** — маршрут посетителя, описание каждой зоны, интерактивные элементы, аудиогид
- **Техническое задание для реализации** — требования к пространству, оборудованию, свету, AV, интерактивным панелям
