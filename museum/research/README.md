# Research — Музейный мониторинг

Ежедневные отчёты Museum Researcher. Формат файлов: `YYYY-MM-DD.md`.

Исследователь мониторит: новые выставки мировых музеев, иммерсивные практики, технологии, поведение аудитории.

## Структура отчёта

Каждый файл `YYYY-MM-DD.md` содержит:

1. **Дайджест** — краткая сводка за период
2. **Выставки** — новые открытия и примечательные проекты
3. **Иммерсивные практики** — кейсы, механики, реакция аудитории
4. **Технологии** — новые решения в музейном дизайне
5. **Аудитория** — исследования поведения посетителей
6. **Выводы для агентства** — операционные рекомендации

## Источники

Artforum, The Art Newspaper, Hyperallergic, MuseumNext, Museums Association, ICOM, Dezeen, Wallpaper*, Artsy, Frieze, The Guardian (art), NYT Arts
